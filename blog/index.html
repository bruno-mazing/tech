Bruno Mazing Blog

Considerations

  • Free to host.
  • Potentially income-generating with subscriptions, or (less preferably) ads
  • Latex support, both inline and for equations.
  • Ease of producing posts and maintenance.

Options

  • Substack. The latex code blocks are very convenient, but inline latex is not easily supported.
  • Medium.
  • Wordpress. Seems like inline latex is well-supported. Haven't tried equations yet though. The code editor mode is nice. Actually, this is looking as the best option so far.
